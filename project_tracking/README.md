# Project Tracking

Folder ini dipakai sebagai pusat dokumentasi hasil per tahap agar progres proyek tidak tercecer dan setiap tahap punya ringkasan yang mudah dibaca.

## Dokumen yang Ada

1. `01_data_collection_summary.md`
   Ringkasan hasil pengumpulan data utama yang sudah dimiliki proyek.
2. `02_scraping_execution_summary.md`
   Ringkasan strategi dan hasil scraping yang benar-benar dijalankan.
3. `03_theme_discovery_summary.md`
   Ringkasan eksplorasi tema data-driven dari post Atsiri.
4. `04_codebook_v1_summary.md`
   Ringkasan penguncian codebook tema final v1.

## Catatan

- Setiap file di folder ini ditulis sebagai summary hasil, bukan checklist status singkat.
- File teknis detail dan CSV bukti tetap berada di folder `data/`.
- Dokumen induk progres tetap ada di `DATA_CONSUMPTION_PROGRESS.md`.
